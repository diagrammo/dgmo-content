# Keyboard Shortcuts

## Global

| Shortcut            | Action                    |
| ------------------- | ------------------------- |
| **⌘ + ,**           | Open Settings             |
| **⌘ + 1**           | Toggle file tree          |
| **⌘ + 2**           | Toggle code editor        |
| **⌘ + Arrow Left**  | Focus previous panel      |
| **⌘ + Arrow Right** | Focus next panel          |
| **⌘ + =**           | Increase font size        |
| **⌘ + -**           | Decrease font size        |
| **⌘ + 0**           | Reset font size           |
| **⌘ + S**           | Save file                 |
| **⌘ + E**           | Export diagram            |
| **⌘ + N**           | New file                  |
| **⌘ + ?**           | Keyboard shortcuts        |
| **⌘ + Shift + F**   | Focus mode (preview only) |
| **⌘ + Q**           | Quit                      |

## File Tree

| Shortcut                | Action                     |
| ----------------------- | -------------------------- |
| **j** / **Arrow Down**  | Move down                  |
| **k** / **Arrow Up**    | Move up                    |
| **l** / **Arrow Right** | Expand folder              |
| **h** / **Arrow Left**  | Collapse folder            |
| **Enter** / **Space**   | Open file or toggle folder |
| **F2**                  | Rename                     |
| **Delete**              | Delete                     |
| **⌘ + D**               | Duplicate                  |

## Editor

| Shortcut          | Action   |
| ----------------- | -------- |
| **Tab**           | Indent   |
| **Shift + Tab**   | Unindent |
| **⌘ + Z**         | Undo     |
| **⌘ + Shift + Z** | Redo     |

## Dialogs

| Shortcut   | Action  |
| ---------- | ------- |
| **Enter**  | Confirm |
| **Escape** | Cancel  |
